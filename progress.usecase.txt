Use Case: Change nutrient elements
Actor(s): User
Brief: 
	User may add and remove nutrient elements and amounts for tracking
Preconditions: The "Progress" page is presented to the user for changing
Post-conditions: Bar chart will update reflecting the new amount of nutrients.
Basic Flow:	
	1) User must enter nutrient name, unit type, and amount into form fields.
	2) User must click "add nutrient" or "remove nutrient" buttons to complete task.
